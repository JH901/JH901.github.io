## < 1. OPIc > - 20201015
> 1.문제 구성
> > 시험시간 40분 (문항당 답변 제한 시간 없음)
> > 주제 5세트 (15문항)
> > 선택형 주제 3개 + 공통형 주제 2개
> > background Survey 주제 최소화 (2강 : 관련 강의)

> 2.총괄적 평가 방식 Holistic Scoring
> > 문제당 개별 배점x, 세트당 등급 부여 방식
> > 총 5개의 세트 등급의 평균값이 본인 등급
> > 주력 주제  vs 비주력 주제 답변 길이 조절
> > 한 가지 소재의 답변 여러 주제 활용
> > 반드시 본인한테 실제 있었던 이야기로 각색해서 답변

> 3.시험 응시 규정
>
> > 재응시 규정 25일, 150에 한번씩 Waiver

> 4.오픽의 평가 기준
> > TEXT TYPE (문장 구조 + 관용구 + 문법  )
> > CONTEXT & CONTENT ( 주제 연관 표현 + 발화량 )
> > TASK & FUNCTION ( 과제 난이도 수행 능력 )
> > COMPREHENSIBILITY ( 강세 + 발음 + 발화속도 )
